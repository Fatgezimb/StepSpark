# Feature Requests

This file is a lightweight backlog for ideas before GitHub issue tracking is fully active.

## Intake Template

### Title

Short feature name.

### Problem

What learner, educator, or operator problem does this solve?

### Proposed Solution

Describe the smallest useful version.

### User Type

Student, tutor, content reviewer, admin, or system operator.

### Learning Value

Explain how this improves recall, recognition, reasoning, retention, or review quality.

### Risks

List medical accuracy, accessibility, privacy, performance, or scope risks.

### Acceptance Criteria

Define how the feature will be judged complete.

## Candidate Ideas

- Instant Recall Card authoring workflow.
- Visual clue highlighting for clinical vignettes.
- High-yield tag taxonomy.
- Review queue based on forgetting risk.
- NBME-style distractor quality checklist.
- Medical citation and review status system.
