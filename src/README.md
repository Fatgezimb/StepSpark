# Source

This folder contains the application shell and design-system code.

## Current Status

The repository now includes:

- `src/app` - application shell only
- `src/design-system` - reusable design-system components, tokens, and stories
- `src/features/instant-recall` - local Instant Recall Card schema, validation, seed data, state hook, tested engine helpers, and feature UI

Do not add database persistence, review engines, API calls, authentication, question generation, or unrelated product workflows here until the relevant architecture decisions are approved.
